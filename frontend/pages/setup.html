Manual Setup

Step-by-step guided configuration

Step 1: Verify MySQL Installation

Check if MySQL is installed by opening a terminal/command prompt and typing:

mysql --version

If not installed, download from mysql.com

Step 2: Start MySQL Server

Ensure MySQL server is running:

Windows: Services → MySQL → Start

Mac: System Preferences → MySQL → Start Server

Linux:

sudo service mysql start

Step 3: Access MySQL

Open MySQL command line:

mysql -u root -p

You'll be prompted to enter your MySQL root password

Step 4: Create Database

Execute these commands:

CREATE DATABASE edudb;
USE edudb;

Step 5: Run Setup Script

From your terminal, navigate to the EduDB directory and run:

mysql -u root -p edudb < database/init_db.sql

This creates all tables and populates sample data

Step 6: Verify Setup

Check that everything is created correctly:

SHOW TABLES;
SELECT COUNT(*) FROM students;

You should see 4 tables and 5 students

MySQL won't start:

Check if another instance is running. Try restarting your computer. Verify port 3306 is not in use.

Access denied error:

Double-check your root password. You may need to reset it using MySQL documentation.

Database already exists:

Drop and recreate: DROP DATABASE edudb; then create again.
